# Markdown Editor - Кроссплатформенный редактор с поддержкой Qt4, Qt5 и Qt6

## Описание
Полнофункциональный Markdown-редактор с режимом WYSIWYG ("что видишь, то и получаешь"), написанием на чистом C++ с использованием фреймворка Qt. Поддерживает версии Qt4, Qt5 и Qt6 благодаря использованию совместимого API.

## Особенности
- **Два режима работы**: Редактор Markdown с подсветкой синтаксиса и визуальный редактор (WYSIWYG)
- **Полная поддержка Markdown**: Заголовки, жирный/курсив/зачеркнутый текст, списки, цитаты, код, таблицы, ссылки, изображения
- **Расширенная работа с таблицами**: Вставка/удаление строк и столбцов
- **Форматирование текста**: Изменение цвета текста, размера шрифта, выравнивание
- **Спецсимволы**: Вставка специальных символов через диалог
- **Проверка орфографии**: Интеграция с Hunspell (русский и английский словари)
- **Экспорт в PDF**: Сохранение документа в формате PDF с сохранением форматирования
- **Гиперссылки и изображения**: Интерактивная вставка с диалогами выбора
- **Справка**: Встроенная система помощи с описанием всех функций

## Требования
### Обязательные
- **Qt**: версия 4.8+, 5.x или 6.x
- **C++ компилятор**: g++, clang++ или MSVC
- **Hunspell** (для проверки орфографии):
  - Ubuntu/Debian: `sudo apt-get install libhunspell-dev hunspell-ru hunspell-en-us`
  - Fedora: `sudo dnf install hunspell-devel hunspell-ru hunspell-en-US`
  - macOS: `brew install hunspell`
  - Windows: Скачать с [официального сайта](https://github.com/hunspell/hunspell/releases)

### Словари
Словари должны быть установлены в системе:
- Русский: `ru_RU.dic` / `ru_RU.aff`
- Английский: `en_US.dic` / `en_US.aff`

## Сборка проекта

### Вариант 1: Использование qmake (рекомендуется для совместимости)

#### Для Qt4/Qt5
```bash
cd /workspace/markdown-editor
qmake markdown-editor.pro
make
```

#### Для Qt6
```bash
cd /workspace/markdown-editor
qmake6 markdown-editor.pro
make
```

Или явно указать путь к qmake:
```bash
/path/to/qt/bin/qmake markdown-editor.pro
make
```

### Вариант 2: Использование CMake (только для Qt5/Qt6)

#### Для Qt5
```bash
mkdir build && cd build
cmake .. -DQT_VERSION_MAJOR=5
make
```

#### Для Qt6
```bash
mkdir build && cd build
cmake .. -DQT_VERSION_MAJOR=6
make
```

## Запуск
После успешной сборки выполните:
```bash
./MarkdownEditor
```

На Windows: `MarkdownEditor.exe`

## Возможности редактора

### Панель инструментов
- **Жирный (B)**, **Курсив (I)**, **Зачеркнутый (S)**
- **Заголовки**: H1, H2, H3
- **Цвет текста**: Выбор цвета из палитры
- **Списки**: Маркированный, нумерованный, задачи (checkbox)
- **Цитата**: Блок цитирования
- **Код**: Строчный код и многострочные блоки кода
- **Ссылка**: Вставка гиперссылок с диалогом
- **Изображение**: Вставка изображений с выбором файла
- **Таблица**: Вставка шаблона таблицы
- **Меню таблиц**: Вставка/удаление строк и столбцов
- **Спецсимволы**: Диалог специальных символов
- **Проверка орфографии**: Кнопка "ABC ✓" (F7)
- **Экспорт в PDF**: Сохранение в PDF

### Горячие клавиши
- `Ctrl+N` - Новый документ
- `Ctrl+O` - Открыть файл
- `Ctrl+S` - Сохранить файл
- `Ctrl+Shift+S` - Сохранить как...
- `Ctrl+E` - Переключение режима (Редактор/Предпросмотр)
- `Ctrl+P` - Печать
- `Ctrl+Shift+E` - Экспорт в PDF
- `F7` - Проверка орфографии
- `F1` - Справка
- `Ctrl+Q` - Выход

### Работа с таблицами
- Вставка новой строки в таблицу
- Вставка нового столбца в таблицу
- Удаление строки из таблицы
- Удаление столбца из таблицы

*Примечание: Для операций с таблицами установите курсор в нужную ячейку.*

## Совместимость с версиями Qt

Проект использует подмножество API Qt, общее для всех версий:
- **Qt4**: Полная поддержка (минимальная версия 4.8)
- **Qt5**: Полная поддержка
- **Qt6**: Полная поддержка

### Особенности реализации совместимости
1. Использованы только стабильные классы и методы, присутствующие во всех версиях
2. Избеганы устаревшие (deprecated) и новые специфичные функции
3. Для работы с цветами текста используется универсальный метод `QTextCursor::mergeCharFormat()`
4. Парсинг Markdown реализован без зависимости от специфичных версий библиотек
5. Проверка орфографии через Hunspell работает независимо от версии Qt
